# PORTFOLIO-TASK-6
This project is a simple and responsive **Personal Portfolio Website** built using **Python Flask**, **HTML**, and **CSS**. The website serves as a digital portfolio where users can showcase their profile, skills, projects, achievements, and provide a way for visitors to get in touch through a contact form
# 🌐 Personal Portfolio Website using Flask

This project demonstrates the fundamentals of web development using Flask, including routing, template rendering, and frontend design.

---

## 🎯 Objective

To develop a personal portfolio website that:

- Displays personal and professional information
- Highlights skills and projects
- Provides a contact form for communication
- Demonstrates Flask-based web application development

## 🚀 Installation and Setup

### Step 1: Clone or Download the Project

Download the project folder and navigate to the project directory.

### Step 2: Install Flask

```bash
pip install flask
```

### Step 3: Run the Application

```bash
python app.py
```

### Step 4: Open in Browser

Visit the following URL:

```text
http://127.0.0.1:5000/
```

---

## 🔄 Application Workflow

1. User opens the portfolio website.
2. Flask handles the request through routing.
3. The `index.html` page is rendered.
4. User can view personal details, skills, and projects.
5. User can submit information through the contact form.
